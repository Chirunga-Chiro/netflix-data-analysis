# Netflix Data Analysis

## Project Overview
This project analyzes Netflix movies and TV shows to uncover trends in content production, genres, and release patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Lab

## Key Insights
- Movies dominate the Netflix catalog.
- The United States produces the most content.
- Drama and International Movies are the most common genres.

## Dataset
Netflix Movies and TV Shows dataset (8800+ titles).
